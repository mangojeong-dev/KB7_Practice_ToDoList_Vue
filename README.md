# Vue TodoList

KB IT's Your Life 7기에서 진행한 Vue 기반 TodoList 예제 toy project입니다.

## 기능
- 할 일 추가
- 할 일 삭제
- 완료 상태 삭제선/상태 변경

## 기술 스택
- Vue
- Vite
- Bootstrap
